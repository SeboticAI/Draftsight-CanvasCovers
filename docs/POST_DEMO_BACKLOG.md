# Post-Demo Backlog

When during the demo build something comes up that's tempting but out of
scope — write it here, then keep moving.

This file exists specifically to absorb scope-creep ideas without losing
them. The discipline is: capture it, defer it, ship the demo.

---

## Format

- One bullet per item
- Brief — one line of what, one line of why if non-obvious
- Tag with `[client-specific]`, `[platform]`, `[ux]`, `[commercial]`, or
  `[engineering]` as appropriate
