Lift Blanket Generator — Quick Start Guide
CanvasCovers add-in for DraftSight · Beta v1.5.0

This add-in turns your paper lift-blanket measurement sheet into a cutter-ready DXF. You type the measurements into a form laid out like the sheet, click Generate, and it draws the cut piece, the COP cut-out, the quilting lines, and the dimensions on the correct cutter layers.

1. Install & turn it on (one-time)

  1. Close DraftSight.
  2. Run BesiaCAD-CanvasCovers-Setup-1.6.0.exe and approve the Windows admin prompt.
  3. Open DraftSight, go to Tools → Add-Ins, and tick CanvasCovers. (Leave "Start Up" unticked.) You only do this once per DraftSight session.
  4. A CanvasCovers tab appears on the ribbon with a Canvas Covers button.

2. Make a drawing

  1. Open or start a drawing first (the tool draws into the active drawing).
  2. Click ribbon → CanvasCovers → Canvas Covers, then pick the Lift Blanket tile.
  3. Fill in the form (next section), then click Generate Drawing.
  4. The view zooms out automatically so the whole drawing (notes included) is visible — no manual zooming needed.
  5. If you ticked the export option, a Save-As box appears to save the DXF (it pre-fills the file name with the blanket text).
Undo: one press of Ctrl+Z removes the entire generated drawing, so it's safe to generate, look, tweak, and re-generate.

3. Filling in the form

Repeating a job

After every drawing you generate, CanvasCovers remembers what you typed. For a batch of identical blankets, click Load Previous Job's Measurements (top of the form) to bring back all the walls and options. Only the order number, network number and project details stay blank — type those fresh, then click Generate.

Project information

Company, Company Initials, AAC Order Number, Network Number, Project, and Date. The AAC Order Number + Company Initials + Network Number become the DXF file name and the printed label on each wall, so fill them in.

Company Name is a drop-down of your regular customers; picking one fills in the initials automatically. You can still type any new name by hand. To add or change customers, edit this file in Notepad:

%AppData%\BesiaCAD\CanvasCovers\customers.csv

(Paste that into the File Explorer address bar.) One customer per line, as Company Name,INITIALS. Save the file, then close and reopen the CanvasCovers form — no DraftSight restart needed.

Wall tabs — Left / Right / Rear

One tab per wall. Each tab shows a diagram of the blanket with the input boxes where the measurements sit on your paper sheet. The diagram is just a guide — it doesn't change shape as you type. The Right wall is shown mirrored from the Left, the way the two walls face each other on the sheet.

FieldWhat to type
DR-L / DR-RThe door-return tabs at the two outer corners. Put 0 if not needed.
Total widthOptional. If filled in, it overrides the five segment boxes for the cut width. The segment boxes can still be used to place the COP.
S1The gap from the door-return line across to the COP.
COP W (S2)The width of the COP cut-out (the middle bottom-row box). This is the COP width — you don't type a separate COP width.
S3The gap from the COP across to the other door-return line.
HeightThe measured height (top of hook / centre of press-stud down to the bottom of the blanket). The tool subtracts the fixing allowance and doubles it for the cut — you type the plain measured number.
COP HThe height of the COP cut-out.
From bottomThe gap from the blanket bottom up to the bottom of the COP.
The top gap (from the top of the COP up to the fold line) is worked out for you and shown next to the COP. It turns red if your COP would cross the fold line — fix the COP height or "from bottom" until it goes back to normal.

If a wall has no COP, untick Include COP cutout on that tab. If you're not doing a wall at all, untick Include this wall.

Layers

The six cutter layers are listed with their colours and which job each one does (Cut / COP / Annotation / Title). The defaults already match your machine — just click Reset to Defaults if anything gets changed by accident:

LayerColourUsed for
1 Rotary BladeBlueThe cut outline (the part that gets cut)
5 Draw and TextMagentaCOP + quilt lines + wall labels (drawn/scored, not cut)
0WhiteDimensions + the project info text

All six layers (0, the four blade layers, and 5 Draw and Text) are created in every DXF so the file is always complete.

4. If the form won't generate

The tool checks your entries and shows a red message listing everything that needs fixing, e.g.:

  • A box left blank or with letters in it.
  • A measured height that isn't bigger than the fixing allowance.
  • A COP that would run past the edge or cross the fold line.
  • No walls ticked, or a layer job not assigned.

Fix the listed items and click Generate again — your entries stay put.

If a drawing looks incomplete the tool will tell you "some entities could not be drawn." If that happens, Undo (Ctrl+Z) and generate again. If it keeps happening on the same job, send us the job details.

  • Are the layers and colours exactly what your machine expects?
